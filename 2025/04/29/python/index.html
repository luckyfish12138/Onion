4.f-string 用来代替输出中的

在Python中，f 前缀用于字符串字面量表示这是一个格式化字符串（formatted string），也被称为f-string。这种字符串允许你在字符串中嵌入表达式，这些表达式会被求值然后替换为它们的结果。

例如，在你的代码片段 print(f"送给小美第{j}只玫瑰花") 中，{j} 是一个占位符，它会被变量 j 的值所替换。这意味着当你运行这段代码时，如果变量 j 的值是 5，那么输出将会是 "送给小美第5只玫瑰花"。

使用f-string是一种简洁且高效的字符串格式化方式，它从Python 3.6开始引入。这种方式比旧的字符串格式化方法（如 % 操作符或 str.format() 方法）更直接，因为它直接在字符串内部进行变量的替换，而不需要额外的函数调用或复杂的格式说明符。

3.1 可变参数

在Python中，*args 语法用于函数定义中，表示该函数可以接受任意数量的位置参数。这里的星号 * 不是乘法运算符，而是一个特殊的符号，用来告诉Python：“把所有传递给这个函数的额外位置参数收集起来，并将它们作为一个元组（tuple）赋值给 args”。

当调用 name("小鬼", "嗷嗷") 时，"小鬼" 和 "嗷嗷" 这两个参数会被打包成一个元组 ("小鬼", "嗷嗷")，然后这个元组被赋值给 args 变量。接着，print(args) 会打印出这个元组，最后 return args 返回这个元组。

*args 的使用非常灵活，因为它允许你创建能够接受不同数量参数的函数，而不需要提前知道会有多少个参数。这在编写需要处理可变数量输入的函数时特别有用。如果你想要接收关键字参数，你可以使用 **kwargs，它会以字典的形式收集所有的关键字参数。

2.迭代器和生成器

2.1 可迭代对象 interable 和迭代器 interator

凡是可以作用于for循环的都属于可迭代对象

凡是可以作用于next()的都是迭代器

含义 ：可以通过 for … in …..这类语句遍历读取数据的对象称之为可迭代对象（Iterable），它可以记住遍历对象的位置，在上次停留的位置继续取做一些事情。

遍历（迭代）：依次从对象中把一个个元素取出来的过程

可迭代对象的条件：

1.对象实现了__iter__()方法

2.__iter__()方法返回了迭代器对象

2.2 生成器

Python 中一边循环一边计算的机制，叫做生成器

定义：通过yield关键字创建的特殊迭代器。

特性：

更简洁的语法实现迭代器

自动保存函数执行状态（局部变量）

支持无限序列生成

迭代器 vs 生成器对比

特性 迭代器 生成器
实现方式 手动编写__iter__和__next__ 使用yield自动生成协议
代码简洁性 较冗长 极简（适合快速实现）
状态保存 需手动维护实例变量 自动保存函数执行上下文
内存占用 低 极低（适合大数据流）
适用场景 复杂迭代逻辑 简单数据流生成

2.2.2 yield 语句

yield语句一次返回一个结果，在每个结果中间，挂起函数，执行next()，在重新从起点挂起点继续往下执行

yield 与 return 对比

特性 yield return
函数类型 生成器函数（返回生成器对象） 普通函数（返回值后终止）
执行状态 暂停并保留状态 终止函数
调用方式 通过next()或send()逐步执行 函数调用直接返回结果
内存占用 极低（按需生成） 可能占用大量内存（全量返回）
